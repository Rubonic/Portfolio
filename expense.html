Expense Tracker

This was a project I started for my MERN full stack course with Coding Dojo.
I wanted to provide a way for individuals to track their expenses and keep track of their budget. This will be a way to keep track of the money spent on different things.

The user will have a detailed view of their expenses and a way to see the total amount spent. The user will also have a way to add new expenses and a way to remove expenses. The user will also have a way to edit their expenses.

Technologies:
  • - Redux Toolkit
  • - RTK Query
  • - Tailwind CSS
  • - MongoDB Atlas
  • - Lodash
  • - Express Framework
