Superbase

This was a project I started for my Python full stack course with Coding Dojo.
I was in the United States Marine Corps; which we only communicated through emails within our occupation. After, getting out and seeing how a company like Amazon uses a chat messaging system for a more direct line of communication.

I decided to create a chat messaging system designed for the Military. The individuals will have to ability to create a account, create chat rooms and message their leaders and companions instantly.

Technologies:
  • - Python
  • - Django
  • - Django REST Framework
  • - React JS
